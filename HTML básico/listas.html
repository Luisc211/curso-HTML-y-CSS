Listas

En HTML tenemos 2 tipos de listas, las ordered lists (ol), en español listas ordenadas, y las unordered lists (ul), en español listas desordenadas, el uso de una o de otra dependara de lo que vayamos a listas, por ejemplo en un paso a paso es recomendable usar ol, y si no importa el orden podemos usar ul, ambas en su interior contienen etiquetas hijas llamadas li, en las cuales se encierran los elementos listados.

Ejemplo de una lista ordenada, u ol

  1. Elemento 01
  2. Elemento 02
  3. Elemento 03
  4. Elemento 04

Ejemplo de una lista desordenada, o ul

  • Elemento 01
  • Elemento 02
  • Elemento 03
  • Elemento 04
