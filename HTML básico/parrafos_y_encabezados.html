La etiqueta h1 representa el título principal de la página, debe usarse solo una vez.

La etiqueta h2 puede representar subtítulos principales y/o secciones importantes dentro de h1.

La etiqueta h3 puede representar subtítulos dentro de h2.

La etiqueta h4 puede representar subtítulos dentro de h3.

La etiqueta h5 puede representar subtítulos dentro de h4.
La etiqueta h6 representa el encabezado de menor nivel; su uso es poco común.
